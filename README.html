Bookmarklets

Info on what a bookmarklet is: http://www.bookmarklets.com/about/

Drag these in to your bookmarks bar if you want to use them.

Create Gmail Links This bookmarklet will change all the mailto links on a page to open up gmail instead of your mail client. Not necessary for Firefox 3+.

Highlight Elements With Style Attribute This will highlight all dom elements on the page that have inline style attributes. Useful when trying to speed up a page.

I have only used these in Safari 4. If anyone uses them in other browsers and have problems lemmie know and I might try and fix em.
