h1. Bookmarklets

Info on what a bookmarklet is: http://www.bookmarklets.com/about/

Drag these in to your bookmarks bar if you want to use them.

"Create Gmail Links":javascript:(function(){var%20script=document.createElement('script'),src=typeof(Sizzle)!='undefined'?'http://github.com/RyanS/Bookmarklets/raw/master/release/bookmarklets.min.js':'http://github.com/RyanS/Bookmarklets/raw/master/release/bookmarklets_with_sizzle.min.js';window.__bf__='g_mailto_links()';script.src=src+'?'+(new%20Date().getTime());document.body.appendChild(script);})(); This bookmarklet will change all the mailto links on a page to open up gmail instead of your mail client. Not necessary for Firefox 3+.

"Highlight Elements With Style Attribute":javascript:(function(){var script=document.createElement('script'),src=typeof(Sizzle)!='undefined'?'http://github.com/RyanS/Bookmarklets/raw/master/release/bookmarklets.min.js':'http://github.com/RyanS/Bookmarklets/raw/master/release/bookmarklets_with_sizzle.min.js';window.__bf__='find_style_attributes()';script.src=src+'?'+(new%20Date().getTime());document.body.appendChild(script);})(); This will highlight all dom elements on the page that have inline style attributes. Useful when trying to speed up a page.

I have only used these in Safari 4. If anyone uses them in other browsers and have problems lemmie know and I might try and fix em.
